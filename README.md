## Running on local
#### 1. Install node.js 
#### 2. clone the repo and install dependencies.
```
git clone https://github.com/jay-repos/react-redux-example react-redux-example
cd react-redux-example
npm install
```
#### 3. start running
```
npm run start
(http://localhost:5000/)
```
